方向

流程图绘画方向

  • TB - 从上而下
  • BT - 从下而上
  • RL - 从右到左
  • LR - 从左到右
  • TD - 从上到下
graph LR
    Start --> Stop

节点

默认节点

graph LR
    id
节点的标识符和内容一致 ### 文本节点 
graph LR
    id1[This is the text in the box]
`[]`,`()`,`(())`,`>]`,`{}`可以声明节点的文本，使得文本和标识符不一致。 其中 - `[]` : 方形 - `()` : 圆方 - `(())` : 圆形 - `>]` : 标签形 - `{}` : 菱形 
graph LR
    id1[This is the text in the box]
    id2(This is the text in the box)
    id3((This is the text in the box))
    id4>This is the text in the box]
    id5{This is the text in the box}

连线

使用标识符进行连线，例如 A 连到 B a --> b
其中

  • --- 实线无箭头
  • --> 实线带箭头
  • -.- 虚线无箭头
  • -.> 虚线带箭头
  • === 粗线无箭头
  • ==> 粗线带箭头

可以在线上加文字，例如 A 连到 B 文字为 你好 a -- 你好 --> b
以Text为例

  • -- Text --- 实线无箭头
  • -- Text --> 实线带箭头
  • -. Text -.- 虚线无箭头
  • -. Text -.> 虚线带箭头
  • == Text === 粗线无箭头
  • == Text ==> 粗线带箭头
